# Scripts por etapa SEMMA

Estructura de trabajo:
- `sample/` -> muestreo e inspeccion local.
- `explore/` -> exploracion descriptiva con Spark.
- `modify/` -> limpieza, tipado y enriquecimiento.
- `model/` -> entrenamiento y comparacion de modelos.
- `assess/` -> evaluacion, graficos y comparacion final.

Ruta canónica recomendada para la entrega:
- `scripts/sample/inspect_sample.py`
- `scripts/explore/explore_puno.py`
- `scripts/modify/modify_puno.py`
- `scripts/modify/preview_modify_puno.py`

Los scripts de `model/` y `assess/` se iran agregando segun el plan de modelado.
